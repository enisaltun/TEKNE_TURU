Modul Sistemi Nasil Calisiyor?

Her modul module_config tablosunda is_active flag'i ile kontrol edilir. Modul kapatildiginda: frontend menuden kaybolur, API endpointleri 404 doner, veritabani RLS politikalari bloke eder. Kod silinmez — istediginiz zaman tekrar acabilirsiniz.
